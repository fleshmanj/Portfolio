The tool is highly customizable, with support for a wide range of transit models and data processing options. The ExoScanner is designed to be flexible and easy to use, with a clear command-line interface and extensive documentation. The tool is well-documented, with detailed instructions for installation, configuration, and usage. The code for the ExoScanner is open-source and available on GitHub under the MIT license. The project is actively maintained, with regular updates and bug fixes, and has a growing community of users and contributors.
